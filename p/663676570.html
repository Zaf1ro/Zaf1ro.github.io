Greatest Common Divisor and Least Common Multiple

01/19/18

1. Greatest Common Divisor

两个或多个整数共有约数中最大的一个. a, b的GCD记为(a，b)

1.1 Euclidean algorithm(欧几里得算法)

欧几里得算法, 又名辗转相除法, 可求最大公约数. 原理: 两个整数的最大公约数等于其中较小的数和两数的差的最大公约数.
证明: 有a, b两个数, 求a和b的最大公约数. a可表示为kb + r, r为a mod b. 假设d为a和b的公约数, 所以d可被a和b整除. 因为r = a - kb, 所以d可以r整除.
举例: a = 1071, b = 462, 求最大公约数: 先从1071中不断减去462直到小于462(1071 = 2 × 462 + 147). 然后从462中不断减去147直到小于147(462 = 3 × 147 + 21). 再从147中不断减去21直到小于21(147 = 7 × 21 + 0). 此时余数是0, 所以1071和462的最大公约数是21.

1.2 Stein algorithm

Euclidean algorithm虽然实现简单, 32位整数的模运算也只需要一个指令周期就可以完成. 但由于现代密码算法的提升, 模运算通常面对1024或更长的数字, 这样模运算的效率就会降低. Stein algorithm抛弃了除法和取模, 从而在大数求GCD方面占有很大优势.
当有两个数a和b时, 算法如下:
a == 0: b为GCD
b == 0: a为GCD
a为偶数且b为偶数: $a = a / 2$, $b = b / 2$
a为偶数且b为奇数: $a = a / 2$
b为偶数且a为奇数: $b = b / 2$
a为奇数且b为奇数: $a = abs(a - b)$, $b = min(a, b)$
原理: $gcd(ka,kb) = k × gcd(a,b)$

2. Least Common Multiple

两个或多个整数共有公倍数中最小的一个. a和b的最小公倍数记为[a，b]

2.1 gcd和lcm的关系

最大公约数和最小公倍数之间之间的关系: 假设两个数a和b, 其最大公约数为c, 最小公倍数为d, 则 a × b = c × d.
证明:

∵ a = c × a1 × a2 × … × an, b = c × b1 × b2 × … × bn
(an和bn分别代表a和b中除c之外的因子, 例如12和16, 其gcd为4, 12的a1 = 3, 16的b1 = 2, b2 = 2)

∴ d = c × a1 × … × an × b1 × … × bn
c × d = c × c × a1 × … × an × b1 × … × bn
a × b = c × c × a1 × … × an × b1 × … × bn

∴ a × b = c × d
