Best Time To Buy And Sell Stock

01/20/18

1. 题目1

1.1 Description

Say you have an array for which the ith element is the price of a given stock on day i. If you were only permitted to complete at most one transaction, design an algorithm to find the maximum profit.
For example, Input: [7, 1, 5, 3, 6, 4], Output: 5 (6 - 1 = 5).

1.2 解法

股票价格数组可转换为第n+1天的价格与第n天的价格之差的数组, 例如: $[7, 1, 5, 3, 6, 4] -> [0, -6, 4, -2, 3, -2]$, 然后求最长子序列, 这时就可使用Kadane’s Algorithm.
Time Complexity: O(n)
Space Complexity: O(1)

2. 题目2

2.1 Description

Say you have an array for which the ith element is the price of a given stock on day i. If You can complete as many transactions as you like. However, you may not engage in multiple transactions at the same time. Design an algorithm to find the maximum profit.

2.2 解法

贪心算法即可, 只要价格为上升趋势就加上差值
Time Complexity: O(n)
Space Complexity: O(1)

3. 题目3

3.1 Description

Say you have an array for which the ith element is the price of a given stock on day i.
Design an algorithm to find the maximum profit. You may complete at most two transactions.

3.2 解法

将prices数组分为两个块, 分别为第一次和第二次交易的范围. 找到这两个范围相加最大的profit.
Time Complexity: O(n)
Space Complexity: O(n)

4. 题目4

4.1 Description

Say you have an array for which the $i^{th}$ element is the price of a given stock on day i.
Design an algorithm to find the maximum profit. You may complete as many transactions as you like with the following restrictions:

  • You must sell the stock before you buy again.
  • After you sell your stock, you cannot buy stock on next day.

4.2 解法

整个程序一共有三种状态:

  1. 买入并持股
  2. 卖出
  3. 冷却(cooldown)

那么三种状态的转换有这么几种情况:

  1. 冷却 -> 买入并持股
  2. 买入并持股 -> 持股
  3. 买入并持股 -> 卖出
  4. 卖出 -> 冷却
  5. 冷却 -> 冷却

所以, 切换到每种状态的公式如下:

  1. 切换到买入并持股: 从冷却切换来 / 由买入并持股切换来
  2. 切换到冷却: 由冷却切换来 / 由卖出切换来
  3. 切换到卖出: 由买入并持股切换来

Time Complexity: O(n)
Space Complexity: O(n)
