50. Pow(x, n)

1. Description

Implement pow(x, n), which calculates x raised to the power n (i.e., $x^n$).

Example:

2. Solution

由于$m^{2n} = m^n \times m^n, m^{2n+1} = m \times m^n \times m^n$. 可分别计算$n \leq 0$和$n < 0$的情况.
需要注意, Integer.MIN_VALUE为$-2^{31}$, 而Integer.MAX_VALUE为$2^{31}-1$, 因此不能直接将负值改为正值.

372. Super Pow

1. Description

Your task is to calculate $a^b$ mod 1337 where a is a positive integer and b is an extremely large positive integer given in the form of an array.

Example:

2. Solution

假设b = $[x, y, z]$, 则计算结果为$a^{x*100+y*10+z}$, 也就是$a^{x*10*10+y*10+z}$, 可转换为$((a^{x})^{10})^{10} \times (a^{y})^{10} \times a^{z}$
