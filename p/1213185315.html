Implement Math.pow()

01/17/18

1. Description

实现Math.pow的功能.

2. 问题点

  1. exponent == 0: 返回1
  2. base == 0 && exponent < 0: 报错
  3. base != 0 && exponent < 0: 取倒数

3. 提高效率

上述算法的时间复杂度为O(n), 这明显不能满足效率. 所以可使用快速幂将时间复杂度降到O(nlogn). 原理如下:
以$2^{11}$为例, 其中$11 = 2^3 + 2^1 + 2^0$, 那么$2^{11} = 2^{2^3} * 2^{2^1} * 2^{2^0}$. 那么

$a^n$ formula
n为偶数 $a^{\frac{n}{2}} * a^{\frac{n}{2}}$
n为奇数 $a^{\frac{n-1}{2}} * a^{\frac{n-1}{2}} * a$

4. 优化

上述代码在普通情况下都可以运行, 但还是存在几点问题:

  1. 当exponent值过大时, 递归调用太多次可能导致StackOverflow
  2. exponent == Integer.MIN_VALUE: 无法获取其正值
