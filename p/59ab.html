97. Interleaving String

1. Description

Given strings s1, s2, and s3, find whether s3 is formed by an interleaving of s1 and s2.

An interleaving of two strings s and t is a configuration where they are divided into non-empty substrings such that:

  • $s = s_1 + s_2 + \ldots + s_n$
  • $t = t_1 + t_2 + \dots + t_m$
  • $|n - m| \leq 1$
  • The interleaving is $s_1 + t_1 + s_2 + t_2 + s_3 + t_3 + \ldots$ or $t_1 + s_1 + t_2 + s_2 + t_3 + s_3 + \ldots$

Note: a + b is the concatenation of strings a and b.

2. Solution

由于s3由s1和s2按顺序组成, 假设dp[i][j]数组表示s1的前i个字符, s2的前j个字符, 组成s3的前i+j个字符. 存在以下几种情况:

  • dp[0][0]一定为true
  • dp[i][0]表示s1的前i个字符与s3的前i个字符相同, dp[0][j]同理
  • 若满足以下情况, 则dp[i][j]为true
    • dp[i][j-1]为true, 且s2[j-1] == s3[i+j-1]
    • dp[i-1][j]为true, 且s1[i-1] == s3[i+j-1]
