Reorder List

03/01/18

1. Description

Given a singly linked list L: $L_0→L_1→…→L_{n-1}→L_n$,
reorder it to: $L_0→L_n→L_1→L_{n-1}→L_2→L_{n-2}→ …$
You must do this in-place without altering the nodes’ values.

For example,
Given {1,2,3,4}, reorder it to {1,4,2,3}.

2. Solution

思路: 该题可拆为三部分:

  1. 将链表拆为两段, 长度相等
  2. 反转第二段链表
  3. 第一段链表和第二段链表交叉连接
