1. Unique Binary Search Trees

1.1 Description

Given n, how many structurally unique BST’s (binary search trees) that store values 1…n?
For example,
Given n = 3, there are a total of 5 unique BST’s.

1.2 Dynamic Programming Solution

思路: 假设已知1个节点所能组成树的组合数为 dp(1) = 1, 2个节点的组合数为 dp(2) = 2, 3个节点的组合数为 dp(3) = 5, 如何求4个节点的组合数:

  • 取一个节点作为树的root
  • 假设root的左子树的节点数为0, 则右字数的节点数为3, 这种情况下的组合数: $dp(0) * dp(3) = 1 * 5 = 5$
  • 假设root的左子树的节点数为1, 则右子树的节点数为2, 这种情况下的组合数: $dp(1) * dp(2) = 1 * 2 = 2$
  • 假设root的左子树的节点数为2, 则右子树的节点数为1, 这种情况下的组合数: $dp(2) * dp(1) = 2 * 1 = 2$
  • 假设root的左子树的节点数为3, 则右子树的节点数为0, 这种情况下的组合数: $dp(3) * dp(0) = 5 * 1 = 5$
  • 则$dp(4) = 5 + 2 + 2 + 5 = 14$
  • 因此: $dp(n) = dp(n - i) + dp(i)$
