Find the single one

01/19/18

1. Description

Given an array of integers, every element appears twice except for one. Find that single one.

1.1 使用HashMap

  1. 遍历整个数组, 统计每个数字的出现频率到HashMap中.
  2. 遍历HashMap, 找出出现一次的数字

Space complexity: O(n)
Time complexity: O(n)

1.2 使用位运算

使用异或运算: a ^ a = 0, a ^ 0 = a
Space compexity: O(1)
Time complexity: O(n)

2. Description

Given an array of integers, every element appears three times except for one, which appears exactly once. Find that single one.

2.1 HashMap

解法与上一题相同
Space complexity: O(n)
Time complexity: O(n)

2.2 位运算(1)

由于是其他数值出现三次, 所以不能单纯的异或运算(a ^ a ^ a = a). 需要统计异或后结果上每一位的个数
Space complexity: O(n)
Time complexity: O(n)

2.3 位运算(2)

  1. 如果数值出现第一次, 将该值保存在ones中
  2. 如果数值出现第二次, 将该值保存在twos中, 将ones中的该值消去
  3. 如果数值出现第三次, 将该值保存在ones中, 将twos中的该值消去
    Space complexity: O(n)
    Time complexity: O(n)

3. 问题描述

Given an array of numbers nums, in which exactly two elements appear only once and all the other elements appear exactly twice. Find the two elements that appear only once.

3.1 HashMap

思路与第一题相同
Space complexity: O(n)
Time complexity: O(n)

3.2 位运算

  1. 假设要寻找的数字为a, b, 那么通过异或整个数组可得到c(c = a ^ b)
  2. 虽然不能通过c直接获得a和b, 但通过c与数组中每个数值的比较可获得a和b
