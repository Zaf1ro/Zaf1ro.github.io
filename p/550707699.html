Remove N-th Node From End of List

02/21/18

1. Description

Given a linked list, remove the $n^{th}$ node from the end of list and return its head.
For example,

2. Solution

思路: 设置两个指针prev和curr, prev表示被删除节点的前一个节点, curr用于遍历整个链表. 先让curr向后推进n-1个节点, 然后prev和curr一起移动, 这样将prev和curr的距离保持一致.

Time Complexity: $O(n)$
Space Complexity: $O(1)$
