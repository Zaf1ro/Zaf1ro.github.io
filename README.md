## SSH
### find ssh location
`where ssh` or `gcm ssh`

### test ssh key
`ssh -T git@github.com`

### debug ssh
`ssh -v hostname`

## Git
### check branch
`git branch`

### swtich to an existing branch
`git checkout master`

### check git username and email
* `git config user.name`
* `git config user.email`

### check git history
`git log`
